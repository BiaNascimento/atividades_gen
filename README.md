# 📚 Atividades Generation - TypeScript & JavaScript

> ⚠️ **Este repositório não é um projeto completo.** É uma coleção de **exercícios resolvidos** desenvolvidos durante estudos de lógica de programação e estruturas de dados com JavaScript e TypeScript.

---

## 📂 Organização dos Exercícios

Cada pasta corresponde a um módulo de estudo e contém os exercícios resolvidos referentes àquele tema.

| Pasta | Tema | Linguagem | Exercícios |
|---|---|---|---|
| `05_operadores` | Operadores aritméticos, relacionais e lógicos | JavaScript | 4 |
| `06_condicionais` | Estruturas condicionais (`if/else`, `switch`) | JavaScript | 8 |
| `07_repeticao` | Laços de repetição (`for`, `while`, `do while`) | JavaScript | 8 |
| `08_collections` | Arrays e coleções de dados | TypeScript | 4 |
| `09_pilha_fila` | Estruturas de dados: Pilha (Stack) e Fila (Queue) | TypeScript | 4 |

## 📁 Detalhes por Módulo

### 05 — Operadores
Exercícios sobre operadores aritméticos (`+`, `-`, `*`, `/`, `%`), operadores relacionais (`>`, `<`, `==`, `!=`) e operadores lógicos (`&&`, `||`, `!`).

### 06 — Condicionais
Exercícios com estruturas de decisão usando `if/else` e `switch/case`, abordando cenários do dia a dia como classificação de dados, validações e menus de opções.

### 07 — Repetição
Exercícios com laços `for`, `while` e `do while`, incluindo problemas de contagem, acumulação, iteração sobre sequencias e controle de fluxo.

### 08 — Collections
Exercícios com arrays em TypeScript, explorando criação, inserção, remoção e busca de elementos em coleções.

### 09 — Pilha e Fila
Implementação e uso prático das estruturas de dados **Stack** (LIFO) e **Queue** (FIFO) em TypeScript, com classes dedicadas (`Stack.ts`, `Queue.ts`) e atividades aplicadas (`atv-stack.ts`, `atv-queue.ts`).

## ▶️ Como executar

### Pré-requisitos

- [Node.js](https://nodejs.org/) instalado
- TypeScript instalado globalmente

```bash
npm install -g typescript
```

### Instalação

```bash
# Clone o repositório
git clone https://github.com/BiaNascimento/atividades_gen.git

# Acesse a pasta
cd atividades_gen

# Instale as dependências
npm install
```

### Rodando um exercício `.js`

```bash
node 05_operadores/01.js
```

### Rodando um exercício `.ts`

```bash
# Compile o arquivo
tsc 08_collections/01.ts

# Execute o arquivo gerado
node 08_collections/01.js
```
